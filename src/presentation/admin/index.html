Annuaire Remote Phonebook UCM

Synchronisez les contacts Odoo dans l'UCM Grandstream pour afficher les noms d'appelants sur les téléphones IP et l'application Wave.

Cette URL est accessible sans authentification afin que l'UCM puisse la lire directement.
Contacts disponibles
Cliquez sur actualiser pour compter les contacts.
Configuration dans l'UCM Grandstream
  1. Connectez-vous à l'interface web UCM → Contacts → Remote Phonebook
  2. Cliquez Add, entrez un nom (ex: Odoo) et collez l'URL ci-dessus dans Phonebook URL
  3. Réglez l'intervalle de synchronisation (ex: Automatique toutes les heures)
  4. Sauvegardez et cliquez Sync pour forcer la première synchronisation
  5. Sur l'application Wave : Paramètres → Contacts → Remote Phonebook
